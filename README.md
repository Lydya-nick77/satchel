# satchel

Satchel is an Ashita v4 addon to replace the ingame inventory, Mog house included.
Welcome to 2026!

## Features

- Container tabs: Inventory, Satchel, Sack, Case, and Wardrobes.
- Per-tab capacity counters (used / total).
- Stack count overlay on item icons.
- Hover tooltip with item description and stats like in game.
- Right click consummables in inventory to use the item.
- Right click gears in inventory or Wardrobes to equip or to use them (RR hairpin for example).
- Gil is now displayed in the window

## Commands

- /satchel 
- /satchel config
