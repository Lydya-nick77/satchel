# satchel

Satchel is an Ashita addon that renders a combined inventory grid UI inspired by the in-game backpack view.

## Features

- Container tabs: Inventory, Satchel, Sack, Case, and Wardrobes.
- Per-tab capacity counters (used / total).
- Stack count overlay on item icons.
- Hover tooltip with item description and stats.

## Commands

- /satchel 
- /satchel config

## Notes

No action possible at this point. This version is only a visual of the different inventories.
